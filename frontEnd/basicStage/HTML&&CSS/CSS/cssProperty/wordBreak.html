word-break属性

word-break: normal | keep-all | break-all;

详解：指定在文本什么位置断词换行。

默认值：normal。
1、对于CJK文字（中文、日文、韩文），可以在任意位置断词换行。
2、对于非CJK文字，只能在空白字符、标点符号处断词换行，否则不换行。

keep-all：在保证单词完整性的前提下，在空白字符或标点符号处断词换行。

break-all：可以在文本任意位置断词换行，不必考虑单词完整性。

- 在html中，空白字符作为单词的分隔符，空格、换行、回车、制表等一个或多个空白字符，在网页中皆显示为一个空格字符，称之为空白折叠现象。

- 空白折叠现象，发生在文本、行盒、行块盒之间。
