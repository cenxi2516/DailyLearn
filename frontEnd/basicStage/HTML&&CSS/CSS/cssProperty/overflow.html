溢出盒子可视区域（填充盒）的内容处理

overflow属性详解

1、overflow是 overflow-x 和 overflow-y 的简写属性。

2、overflow-x 和 overflow-y 的初始值皆为 visible。

3、overflow的属性值：visible | hidden | auto | scroll | clip

  1. visible：对于溢出盒子填充盒的内容，不做任何处理，可能会溢出盒子。
  2. hidden：对于溢出盒子填充盒的内容，直接隐藏，不显示滚动条。但可以通过编程，让盒子滚动。
  3. auto：根据是否存在有溢出盒子填充盒的内容，按需在填充盒显示水平或垂直方向上的滚动条。
  4. scroll：不管是否存在有溢出盒子填充盒的内容，皆在填充盒显示水平和垂直方向上的滚动条。
  5. clip：对于溢出盒子填充盒的内容，直接隐藏，不显示滚动条。完全禁止盒子滚动，通过编程也无法让盒子滚动。

4、overflow适用于块盒。

5、overflow-x，用于处理水平方向上溢出盒子填充盒的内容。

6、overflow-y，用于处理垂直方向上溢出盒子填充盒的内容。

7、若是设置overflow-x(overflow-y)，则浏览器强制将overflow-y(overflow-x)设置为auto。

  1. 若是用户手动再设置overflow-y为visible，则浏览器也会强制设置为auto。
  2. 若是用户手动再设置overflow-y为非visible，则为用户设置的值。

8、若是盒子设置overflow或overflow-x或overflow-y的属性值为非visible，则会触发盒子的BFC。

9、若是table元素的table-layout设置为fixed，则th、td元素可以使用overflow属性，来处理溢出盒子填充盒的内容。
