content属性

1、content属性只能在::before和::after伪元素中使用。
2、为::before和::after伪元素插入内容。
3、content属性值：
   a、normal：初始值，等价于none。
   b、none：不产生伪元素。
   c、<string>：文本内容。
   d、url(uri)：返回uri图片资源。
   e、attr(x)：返回元素中x属性的字符串内容。
   f、open-quote：左侧引号。
   g、close-quote：右侧引号。
