# body背景
> **画布 canvas**：是一块区域。
**特点**：
- 画布的最小宽度为视口宽度。
- 画布的最小高度为视口高度。


## html元素的背景
> 覆盖画布的背景。


## body元素的背景
- 若是html元素存在背景，则body元素背景正常显示。
- 若是html元素不存在背景，则body元素背景覆盖画布。

### body元素的背景图
- 背景图的宽度为百分比，相对于视口的宽度。
- 背景图的高度为百分比，相对于html元素的高度。

- 背景图的横向坐标为百分比、预设值，相对于视口的宽度。
- 背景图的纵向坐标为百分比、预设值，相对于html元素的高度。
